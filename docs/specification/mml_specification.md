# Music Macro Language (MML) 仕様書

本ドキュメントでは `upy_light_engine` でサポートされている MML (Music Macro Language) の仕様について解説します。
本エンジンの MML パーサ (`engine/mml_parser.py`) は軽量ながら、タイ（`&`）やカスタム波形（`@`）などゲーム開発に必要な拡張機能をサポートしています。

## 基本構造
- **大文字・小文字の区別なし**: すべてのコマンドは小文字・大文字どちらでも記述可能です。
- **空白の無視**: スペース、タブ、改行文字はパース時に無視されるため、読みやすくフォーマットできます。
- **マルチトラック**: 現在は複数のトラック（和音や並行パート）をカンマ `,` で区切って1つの文字列として記述します。
  - 例: `CDEFG, EFGAB, @3C4` (3トラック構成)
  - **【今後の課題】**: カンマ区切りの単一文字列では、長大な楽曲データになった際に人間やAIにとって非常に可読性が悪くなります（各トラックの長さを揃えたり、コメントを入れたりするのが困難）。将来的には「トラックごとに配列（リスト）を分ける」「チャンネルごとの辞書型にする」など、楽譜表現（データ構造）の仕様を見直す予定です。

## 基本コマンド一覧

| コマンド | 説明 | デフォルト値 | 例 |
|---|---|---|---|
| `T<num>` | **テンポ (Tempo)**: BPM（1分間の四分音符の数）を設定します。トラックをまたいで引き継がれます。 | 120 | `T150` (BPM150) |
| `O<num>` | **オクターブ (Octave)**: 基準となるオクターブ(0〜8)を設定します。 | 4 | `O5` |
| `L<num>` | **音符のデフォルト長 (Length)**: 長さが省略された音符や休符のデフォルト長（全音符を1とした分母）を設定します。 | 4 (四分音符) | `L8` (八分音符) |
| `V<num>` | **ボリューム (Volume)**: 音量を設定します。 | 64 | `V100` |
| `@<num>` | **波形タイプ (Wave Type)**: 再生するオシレータの波形を選択します。 | 0 | `@2` (三角波) |
| `<` | **オクターブダウン**: オクターブを1下げます（下限 0）。 | - | `<` |
| `>` | **オクターブアップ**: オクターブを1上げます（上限 8）。 | - | `>` |
| `$` | **ループマーカー (Loop Marker)**: この位置から曲をループ再生します。指定しない場合は曲全体をループするか、1回のみ再生されます（再生環境による）。 | - | `$` |

## 音符と休符コマンド

### 音符 (C, D, E, F, G, A, B)
音階を指定して発音します。
- **半音**: 音名の直後に `#` または `+` で半音上げ（シャープ）、`-` で半音下げ（フラット）を指定できます。（例: `C#`, `B-`）
- **長さ**: 音名の直後に数字をつけることで、その音符だけの長さを指定できます。（例: `C8`, `D#16`）
- **付点**: 数字の直後（または省略時は音名の直後）に `.` をつけると付点音符（長さ1.5倍）になります。（例: `C4.`, `E.`）

### 休符 (R)
無音の区間を指定します。
- 音符と同様に、長さや付点の指定が可能です。（例: `R8`, `R2.`）

### タイ (&)
同じ高さの音符（または休符）の長さを加算して繋げます。
- **使用方法**: `音符1 & 音符2` のように、繋げたい音符の間に `&` を記述します。
- エンジン側では2つの音符イベントとしてではなく、合算された長さを持つ1つの音符として処理（Gapless）されます。
- 例: `C4 & C8` (四分音符と八分音符の長さ分だけ、一度のCの発音を維持する)

## 波形タイプ（`@` コマンド）の仕様

| 指定値 | 波形の種類 | 特徴・用途 |
|:---:|---|---|
| `@0` | **Square (矩形波)** | デフォルトの波形。ファミコン的な電子音。メロディに向いています。 |
| `@1` | **Sawtooth (ノコギリ波)** | 鋭く豊かな倍音を持つ波形。ブラス系の音やベースに。 |
| `@2` | **Triangle (三角波)** | 倍音の少ない丸い音。ベースラインや柔らかな和音に。 |
| `@3` | **Noise (ノイズ)** | ホワイトノイズ。爆発音や打楽器（ハイハット、スネア）などの効果音（SE）に。 |

## 記述例
```mml
T140 V80 O4
@0 L8 C D E F G4. & G8
@2 < C > C < G > G
@3 L16 C C C C R8 C8
```
